# See the List of Products in Cart

See the List of Products That Have Been Added to the Shopping Cart

**URL** : `/api/cart/products`

**Method** : `GET`

## Success Response

**Code** : `200 OK`

**Content examples**

```json
[
    {
        "id": 47,
        "name": "Shoes"
    },
    {
        "id": 81,
        "name": "Sandals"
    }
]
```
