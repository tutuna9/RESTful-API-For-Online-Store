# View Product List By Product Category

Get the list of products based on their category

**URL** : `/api/products/{id}`

**Method** : `GET`

## Success Response

**Code** : `200 OK`

**Content examples**

Product with a CategoryId of 2

```json
{
    "categoryid": 2,
    "id": 1,
    "name": "Bag"
}
```
